Pago en una página del sitio web

Pago completo en una página

El pago de una sola página de Odoo facilita a los clientes hacer los pedidos más rápido al integrar todos los pasos de pago en una página y eliminar la redirección innecesaria de una página a otra. El cliente puede completar la información de facturación, envío y pago en la misma página y realizar el pedido en segundos.

Información

Guía del usuario

El pago en una página del sitio web proporciona una función al cliente, el pago en una página significa que no hay necesidad de perder tiempo en la redirección en muchas páginas para el pago.


El administrador puede personalizar completamente el pago de una página del sitio web, el administrador puede elegir la función de pago de una página habilitada o deshabilitada.

El administrador también puede personalizar el nombre del panel y mostrar la propiedad de ocultar del panel significa que si el administrador no quiere mostrar ningún panel, el administrador puede ocultar fácilmente ese panel. El administrador puede establecer campos obligatorios para facturación y dirección de envío según la necesidad y el tipo de producto.


Principalmente, hay tres paneles en la página de pago. El usuario puede completar fácilmente la página y, al hacer clic en el botón Continuar, complete el proceso de pago.
