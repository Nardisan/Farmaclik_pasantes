Website Membership Program

A program that enables prime membership features on your website.

Few Words About The Website Membership Program

This module gives you a flexibility to create memberships plan and their benefits for website customer and helps you to retain customers for a long period. It will build a relationship between you and your customers.


Salient Features:

  • Easily create multiple membership plans
  • Membership Plan for multiple time duration
  • Manage product's prices for customers with different membership plans
  • Price Management using Odoo Pricelists
  • Separate Website Page to show the all membership plans, you can easily publish or un-publish any plan
  • You can also provide delivery (shipping) offers for customers with membership
  • Set Exclusive Products for membership customers. Only Customer with a valid membership plan can buy those products
  • Control the exclusions of products by time
  • More customizations available...
